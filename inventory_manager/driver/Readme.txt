*******************************************************************************************
				How to Install StockManager
*******************************************************************************************

手順1.	"https://java.com/ja/"からJAVAをインストールする。
	本ソフトはJava開発のため、インストールされてないと動きません。
	※インストールしたディレクトリを覚えておいてください。

手順2.	"./Phidget22-x64_1.0.0.20170906.exe"をインストールする。
	これをインストールしないと、RFIDのドライバではじかれます。
	※Windows 64bitにしかインストールできません。

手順3.	"./phidgets/phidget22.jar"を"Java/jre1.8.0_xxx/lib/ext/"と"Java/jdk1.8.0_xxx/jre1.8.0_xxx/lib/ext/"直下にコピーして下さい。
	これを入れないと、RFIDを操作できません。


手順4.	"./connector-j/mysql-connector-java-5.1.43-bin.jar"を"Java/jre1.8.0_xxx/lib/ext/"と"Java/jdk1.8.0_xxx/jre1.8.0_xxx/lib/ext/"直下にコピーして下さい。
	これを入れないと、MySQLサーバに接続できません。
